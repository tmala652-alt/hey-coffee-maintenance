## 3. การแจ้งซ่อม (สำหรับพนักงานสาขา)

### ขั้นตอน
```
1️⃣ คลิก "แจ้งซ่อมใหม่"
   ↓
2️⃣ เลือกสาขาและตำแหน่ง
   ↓
3️⃣ พิมพ์หัวข้อ เช่น "แอร์ไม่เย็น"
   ↓
4️⃣ เลือกประเภท: ⚡ ไฟฟ้า | 💧 ประปา | ❄️ แอร์ | ☕ เครื่องชง
   ↓
5️⃣ เลือกความสำคัญ: 🟢 ต่ำ | 🟡 ปานกลาง | 🟠 สูง | 🔴 เร่งด่วน
   ↓
6️⃣ ตั้ง SLA: 4h | 8h | 24h | 48h | 72h
   ↓
7️⃣ กรอกรายละเอียด + แนบรูป (ถ้ามี)
   ↓
8️⃣ ส่งแจ้งซ่อม ✅
```

---

## 4. การจัดการงาน (สำหรับช่าง)

### Flow การทำงาน
```
📋 รับงาน → 🔧 เริ่มงาน → ⏸️ หยุดพัก (ถ้าจำเป็น) → ▶️ ดำเนินต่อ → ✅ เสร็จสิ้น
```

### ปุ่มลัด
| ปุ่ม | การใช้งาน |
|------|----------|
| 🟢 เริ่มดำเนินการ | เริ่มทำงาน |
| 🟡 หยุดพัก | หยุดชั่วคราว (SLA หยุดนับ) |
| 🔵 ดำเนินการต่อ | กลับมาทำงาน |
| ✅ เสร็จสิ้น | จบงาน |

---

## 5. การดู SLA

### ความหมายสี
```
🟢 เขียว  = ปกติ (0-74%)
🟡 เหลือง = เตือน (75-89%)
🟠 ส้ม    = วิกฤต (90-99%)
🔴 แดง   = เกินกำหนด (100%+)
```

### ตัวอย่างการแสดง
```
┌────────────────────────┐
│ SLA: 4h               │
│ เหลือ: 2h 30m          │
│ [██████░░░░] 62%      │
│ 🟢 On Track           │
└────────────────────────┘
```

## 6. การแจ้งเตือน

### ไอคอนกระดิ่ง 🔔
- ตัวเลขสีแดง = จำนวนที่ยังไม่อ่าน
- คลิกเพื่อดูรายละเอียด

### ประเภทแจ้งเตือน
| Icon | ความหมาย |
|------|----------|
| 🔧 | ได้รับงานใหม่ |
| ⏰ | SLA ใกล้หมด |
| ✅ | สถานะเปลี่ยน |
| 💬 | มีแชทใหม่ |

## 8. ทางลัดที่ควรรู้

| การกระทำ | ทางลัด |
|----------|--------|
| แจ้งซ่อมใหม่ | คลิกปุ่มบนแบนเนอร์ Dashboard |
